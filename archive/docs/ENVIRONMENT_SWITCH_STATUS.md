## 使用方法

### 1. 配置测试环境API密钥

用户需要：

1. 访问 https://testnet.binancefuture.com
2. 注册测试账号
3. 生成API密钥
4. 更新 `.env` 文件

### 2. 启动机器人

```bash
./run.sh
```

### 3. 使用环境切换功能

1. 在Telegram中发送 `/panel` 命令
2. 点击环境按钮（显示当前环境状态）
3. 选择要切换的环境
4. 如果是切换到生产环境，需要确认

## 安全特性

1. **环境隔离**: 测试环境和生产环境使用不同的API密钥
2. **安全确认**: 切换到生产环境需要二次确认
3. **密钥验证**: 切换到生产环境前验证主网API密钥是否存在
4. **环境信息显示**: 面板显示当前环境类型和API密钥来源
5. **交易安全**: 两个环境的持仓完全隔离
